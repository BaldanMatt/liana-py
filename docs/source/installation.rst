Installation
============

**liana** requires Python version >= 3.8 to run.

Development Version
-------------------

To install the most up-to-date version, run:

.. code-block:: console

   pip install git+https://github.com/saezlab/liana-py


Stable Version
-------------------

To install the latest stable version from pip, run:

.. code-block:: console

   pip install liana
